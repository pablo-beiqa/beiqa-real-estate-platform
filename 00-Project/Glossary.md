# Glosario - BEIQA Platform

## Términos de Negocio

| Término | Definición |
|---------|------------|
| **Tenant** | Empresa cliente de Beiqa que busca espacio (no confundir con tenant de software multi-tenant) |
| **Tenant Representation** | Servicio de representación donde Beiqa actúa en nombre del arrendatario, no del propietario |
| **Deal** | Transacción inmobiliaria desde solicitud inicial hasta cierre |
| **Shortlist** | Lista de propiedades preseleccionadas para presentar a un cliente |
| **Comps (Comparables)** | Propiedades similares usadas para comparar precios y condiciones |
| **Vacancy Rate** | Tasa de vacancia - porcentaje de espacio desocupado en un mercado |
| **Absorción** | Cantidad de espacio ocupado en un período (m² rentados - m² desocupados) |

## Tipos de Inmuebles

| Término | Definición |
|---------|------------|
| **Nave Industrial** | Edificio para manufactura, almacenamiento o distribución |
| **Bodega** | Espacio de almacenamiento, generalmente más pequeño que nave |
| **Centro de Distribución** | Instalación logística para recepción y envío de mercancías |
| **Parque Industrial** | Zona con múltiples naves/bodegas e infraestructura compartida |
| **Retail** | Espacio comercial para venta al público |
| **Oficinas** | Espacio de trabajo administrativo |
| **Build-to-Suit (BTS)** | Construcción a medida según necesidades del cliente |

## Términos Geográficos

| Término | Definición |
|---------|------------|
| **AGEB** | Área Geoestadística Básica - unidad de INEGI más pequeña que municipio |
| **Código Postal (CP)** | Código de 5 dígitos de SEPOMEX |
| **Colonia** | División urbana dentro de municipio |
| **Municipio** | División administrativa dentro de estado |
| **Polígono** | Área geográfica definida por coordenadas |
| **Isócrona** | Área alcanzable desde un punto en tiempo determinado |

## Términos Técnicos

| Término | Definición |
|---------|------------|
| **Scraping** | Extracción automatizada de datos de sitios web |
| **API** | Application Programming Interface - forma de conectar sistemas |
| **GeoJSON** | Formato de datos geográficos basado en JSON |
| **Shapefile** | Formato de datos vectoriales de ESRI |
| **KML** | Keyhole Markup Language - formato de datos geográficos de Google |
| **PostGIS** | Extensión de PostgreSQL para datos geoespaciales |
| **ETL** | Extract, Transform, Load - proceso de movimiento de datos |
| **Geocoding** | Convertir direcciones a coordenadas geográficas |

## Fuentes de Datos

| Término | Definición |
|---------|------------|
| **INEGI** | Instituto Nacional de Estadística y Geografía |
| **DENUE** | Directorio Estadístico Nacional de Unidades Económicas |
| **datos.gob.mx** | Portal de datos abiertos del gobierno mexicano |
| **Catastro** | Registro de propiedades con valores y características |
| **SCIAN** | Sistema de Clasificación Industrial de América del Norte |

## Métricas Inmobiliarias

| Término | Definición | Fórmula/Ejemplo |
|---------|------------|-----------------|
| **Precio por m²** | Costo por metro cuadrado | Precio total / m² totales |
| **Cap Rate** | Tasa de capitalización | NOI / Valor de propiedad |
| **NOI** | Net Operating Income | Ingresos - Gastos operativos |
| **Yield** | Rendimiento | Renta anual / Precio de compra |
| **Vacancy Rate** | Tasa de vacancia | m² vacantes / m² totales |
